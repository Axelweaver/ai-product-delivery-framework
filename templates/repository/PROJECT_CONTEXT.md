# PROJECT_CONTEXT.md

## Purpose

This document provides a high-level overview of the current project.

Its primary audience is AI systems and new contributors.

The goal is to allow someone unfamiliar with the repository to quickly understand:

- what the project is;
- why it exists;
- what has already been completed;
- what is currently in progress;
- what should happen next.

This document should remain concise.

Detailed information belongs in dedicated documentation.
